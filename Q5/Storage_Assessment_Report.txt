6. RECOMMENDATIONS FOR IMPROVING STORAGE UTILIZATION
----------------------------------------------------

Recommendation 1 - Enable Log Rotation:
  Configure logrotate to manage /var/log growth:
    vi /etc/logrotate.d/custom
    (add: /var/log/*.log { weekly rotate 4 compress })
  Impact: Prevents unbounded log growth; can save 50-200 MB/month.

Recommendation 2 - Add Swap Space:
  No swap detected; protect against OOM events:
    fallocate -l 4G /swapfile
    chmod 600 /swapfile
    mkswap /swapfile
    swapon /swapfile
    echo '/swapfile swap swap defaults 0 0' >> /etc/fstab
  Impact: System stability under memory pressure.

Recommendation 3 - Clean APT Package Cache:
  Reclaim space from cached package files:
    apt-get clean
    apt-get autoremove
  Impact: Can reclaim 1-3 GB from /var/cache/apt/.

Recommendation 4 - Set Up Disk Monitoring Alert:
  Alert when disk exceeds 80%:
    df -h | awk 'NR>1 && int($5) > 80 {print "ALERT: "$0}'
  Impact: Early warning prevents service-disrupting full-disk events.

Recommendation 5 - Archive and Compress Old Data:
  Compress infrequently accessed files:
    find /var/log -name "*.log" -mtime +30 | xargs gzip -v
  Impact: Text logs compress 5-10x, recovering significant space.

Recommendation 6 - Monitor Inode Consumption Rate:
  Track inode-heavy directories:
    find / -maxdepth 4 -xdev 2>/dev/null | awk -F/ '{print NF, $0}' | sort -rn | head -20
  Impact: Identify small-file generators before exhaustion occurs.
